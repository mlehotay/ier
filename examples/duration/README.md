# Duration Example Set

## Status

Status: reference  
Authority: non-canonical  

This directory contains an exploratory application of Informational Experiential Realism (IER) to contemporary neuroscience research on temporal phenomenology and duration categorization.

The primary article is [Duration Without Clocks](IER-duration-without-clocks.md).

It reinterprets a recent cortical timing study through the IER framework, arguing that the observed cortical hierarchy may be better understood as organization of:

* admissible continuation,
* continuation topology,
* coherence-preserving futures,
* and event articulation,

rather than representation of elapsed time by an internal clock.

The accompanying [Neuroscience Primer](neuroscience-primer.md) is intended for
readers familiar with IER but not with neuroscience terminology or cortical
organization.


## Scope

This example set is:

* explanatory,
* exploratory,
* and non-canonical.

It introduces:

* no new ontological primitives,
* no criteria for experience,
* no diagnostics,
* and no modifications to the IER identity claim.

The purpose of these documents is not to prove IER empirically.
Their purpose is narrower:

> to demonstrate how the IER framework can reorganize interpretation of temporal phenomenology and cortical timing research.


## Relationship to the Core Framework

The duration reinterpretation depends on existing IER structural machinery developed elsewhere in the corpus, including:

* intrinsic constraint,
* admissible futures,
* continuation topology,
* integration envelopes,
* event segmentation,
* continuity,
* and subjective time.

For this reason, the repository includes a supporting subset of IER articles alongside this example set.

These supporting documents remain in the source corpus and should be treated as
the authoritative source locations.

## Interpretation Boundary

Nothing in this example set should be interpreted as claiming:

* that neuroscience proves IER,
* that cortical regions literally instantiate formal operators,
* that UEFs can be localized neurally,
* or that temporal phenomenology reduces to fMRI activity.

The claim is interpretive and structural:

> the empirical organization identified in the referenced neuroscience study may be more coherently understood as organization of admissible continuation and event articulation than as representation of elapsed time by an internal clock.
